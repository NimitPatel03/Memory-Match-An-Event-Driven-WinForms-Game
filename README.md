# Memory-Match-An-Event-Driven-WinForms-Game
project
Memory Match is a high-performance desktop application built using C# and the .NET Framework. The application features a classic memory-testing interface where users must identify pairs of matching icons hidden within a dynamic 4x4 grid. The project focuses on implementing robust event-driven architecture, asynchronous UI management, and efficient in-memory state tracking. By prioritizing a "Managed Code" approach, the application ensures a smooth user experience with a minimal memory footprint.
